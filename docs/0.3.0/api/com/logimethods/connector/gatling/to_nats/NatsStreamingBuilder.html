com.logimethods.connector.gatling.to_nats

NatsStreamingBuilder

Related Doc: package to_nats

case class NatsStreamingBuilder(messageProvider: AnyRef) extends ActionBuilder with Product with Serializable

A Gatling ActionBuilder to inject messages into NATS.

Possible usage:

val natsScn = scenario("NATS call").exec(NatsStreamingBuilder(new ValueProvider()))
class ValueProvider {
  val incr = 10
  val basedValue = 100 -incr
  val maxIncr = 50
  var actualIncr = 0

  override def toString(): String = {
    actualIncr = (actualIncr % (maxIncr + incr)) + incr
    (basedValue + actualIncr).toString()
  }
}
messageProvider

the provider of the messages to emit. The actual message will be the output of the toString() method applied to this object (which could be a simple String if the message doesn't have to change over time).
